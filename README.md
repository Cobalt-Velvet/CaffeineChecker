# Caffeine Checker ☕️

> 東洋大学 総合情報学部 総合情報学科 システム情報専攻 2年  
> **アプリ制作基礎 最終課題**

日々のカフェイン摂取量を管理・追跡する iOS アプリケーションです。  
過剰摂取を防ぎ、健康的なカフェイン摂取をサポートします。

## 主要機能

- **プリセットドリンク追加** : アメリカーノ、ラテ、エナジードリンクなど 9 種類から選択
- **オリジナルブレンド作成** : 材料を組み合わせてオリジナルドリンクのカフェイン量を計算
- **視覚的な摂取量表示** : プログレスバーと色分けで現在の摂取状況を一目で確認
- **摂取履歴記録** : 今日飲んだドリンクの一覧表示
- **ハプティックフィードバック** : 操作時の心地よい触覚演出

---

## カフェイン摂取レベル

| 摂取量 | 状態 |
|--------|------|
| 0mg | まだ摂取していません |
| 1 ~ 99mg | 適度な摂取量 ✅ |
| 100 ~ 199mg | 良いコンディション 💪 |
| 200 ~ 299mg | 少し多くなっています ⚠️ |
| 300 ~ 399mg | 摂取量が多いです 🚨 |
| 400mg~ | カフェイン過多 🛑 |
